哔——「不吐不快」(ButuBuKuai) 🤬➡️🎵
你的直播间物理级防封号外挂 | 基于阿里云大模型 & OBS WebSocket 的智能消音导播台

💡 为什么做这个？
直播时代，一句话说错，可能意味着几个月的努力付诸东流。
市面上的消音软件要么延迟极高，要么设置反人类，要么会产生严重的系统回音。
「不吐不快」诞生于一个极客的强迫症：我要让直播间的消音，像大厂专业导播台一样毫秒级精准、原生双轨发声，而且小白必须 3 分钟就能上手！

🚀 核心黑科技 (Features)
🧠 毫秒级云端大脑：接入阿里云 DashScope (Paraformer-realtime-v2) 流式语音识别，极速捕捉每一句高危发言。

🎛️ OBS 原生双轨无缝剪切：告别渣音质和桌面回音！直接劫持 OBS 底层 WebSocket 协议，闭麦瞬间触发原生媒体源播放（支持自定义各种搞笑美化音/哔哔声）。

🎯 本地声学精准试音舱：首创“所听即所得”校准法。不用开直播，录音 5 秒，拉动微调滑块，立刻听见防封音效是否完美覆盖！

⚡ 工业级 AOT 词池编译：彻底摒弃低效遍历，词池宏指令（如 #城市#、#带货词#）在启动时执行先行平铺编译，匹配复杂度仅为 O(1)。

👶 保姆级傻瓜 UI：单页化全景布局，视觉降噪设计，所有配置一目了然。

🎬 3 分钟开播指南 (快速上手)
注意： 首次使用，请确保你已经安装了 OBS Studio。

下载与运行：前往 Releases 下载最新的单文件 .exe。

拿“钥匙”：点击获取 阿里云 DashScope API Key，填入软件。

设外挂：在 OBS 中添加名为 美化音轨道 的媒体源。并在高级音频属性中，将麦克风的同步偏移设为 2000ms。（这是让子弹飞一会儿的物理外挂！）

一键接头：点击软件中的 [连通 OBS] 与 [连通 AI 大脑]。

靶场试音：在左下角[精准校准舱]录音5秒。听回放调滑块，直到哔声盖住脏话，保存收工！

如何在本地跑起来？
Fork 本仓库并 Clone 到本地。

复制根目录下的 appconfig.example.json 并重命名为 appconfig.json。

使用 Visual Studio 2022 打开项目，或直接在根目录运行 dotnet run。

根目录下自建 HelpImages 文件夹，放入 step1.png ~ step5.png 即可激活图文帮助。
