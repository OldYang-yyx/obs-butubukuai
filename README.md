# obs-butubukuai
「不吐不快」- 一款为国内直播生态定制的开源 OBS 违禁词实时自动屏蔽/消音插件。支持自定义屏蔽音，让你在直播间安心输出。

「不吐不快」 (Butubukuai) - 你的直播言论护航员
1. 项目简介

核心口号：既然不吐不快，那就畅所欲言。

这是一个专为国内直播生态设计的开源 OBS 插件。通过 AI 实时流式识别，在违禁词触达观众前，精准实施“闭麦+消音”操作。

2. 核心逻辑 (必读)

时间差拦截技术：本插件通过监听实时麦克风，在 OBS 的推流延迟窗口内完成识别与动作。建议用户在 OBS 中为麦克风设置 2000ms 以上的同步偏移。

3. 功能特性

🚀 极速响应：基于阿里百炼 Paraformer-V2 实时语音模型，毫秒级转写。

🛠️ 一键联动：支持 OBS WebSocket 5.x 协议，自动识别音轨，无需手动输入名称。

🔇 智能防抖：命中违禁词后自动闭麦并播放屏蔽音，支持连续输出下的封禁自动重置。

📂 配置持久化：本地保存 API Key 与自定义词库。
